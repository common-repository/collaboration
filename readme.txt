A collaboration tool to integrate TogetherJS in to WordPress.

== Description ==

This plugin adds a menu item to the admin bar that reads "Collaborate" that when clicked launches a TogetherJS instance.

There are various settings that allow you to configure the TogetherJS options to your liking.

Let me know what you would like to see in future versions.

What other functionality should be implemented to help you collaborate with others?

== Installation ==

Drop into either your /mu-plugins/ directory, or your plugins directory and activate it.

1. Upload the `collaboration` directory to the `/wp-content/plugins/` directory, or the `/wp-content/mu-plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress (if placed in the the `/wp-content/plugins/` directory).
1. You're set! If you want to change settings, do so in "Settings > Collaborate."
